# report: 52週レンジの髭(AdjH/AdjL)ベース統一＋正典明記(2026-07-07)

## 背景
前タスク(d2a2f9b、株価系列の分割調整パッチ)の検証時、52週レンジの算出基準が過去の手動補正間で不統一だったことが露見: 1980ダイダンのノートは「終値(AdjC)ベース」で手動再計算していたが、3110日東紡績のノートは「高値/安値(AdjH/AdjL、髭)ベース」で手動再計算していた。ケンジ判断により「髭ベース」に正典統一する(株探等の外部表示・実際のローソク足の高値/安値と照合可能なprovenance重視)。

## 変更内容

### タスクA: fetch_stock.py
- 52週高値を`AdjH`(調整後高値)の期間max、52週安値を`AdjL`(調整後安値)の期間minに変更(対応する日付も高値/安値発生行から取得)
- EMA13/25/75・現在値・出来高は`AdjC`/`AdjVo`(終値・調整後出来高)ベースのまま**変更していない**
- `price.note`を「株価系列は分割調整後。52週レンジは高値/安値(髭)ベース、EMAは終値ベース」に更新

### タスクB: 正典明記(SKILL.md v5.5.7→v5.5.8)
コア⑧局面分岐の手順に「52週レンジ・スイングアンカーは髭(AdjH/AdjL、分割調整後)ベースを正典とする。終値ベースを使う場合は例外として理由を明記」を追記。冒頭バージョン表記・改訂記録も整合。

## 検証結果
- **3110(必須検証)**: 修正後の52週高値6,580円(2026-05-07)・安値1,050円(2025-08-20)が、既存ノート記載の手動髭ベース再計算値と完全一致
- **サニティ(4銘柄)**: 1980/3110/6637/9616のいずれも「髭高値≥終値高値」「髭安値≤終値安値」が成立(列取り違えなし)
  - 1980: 髭高値3,680 vs 終値高値3,670 / 髭安値1,433.3 vs 終値安値1,443.3
  - 3110: 髭高値6,580 vs 終値高値6,390 / 髭安値1,050 vs 終値安値1,060
  - 6637: 髭高値5,370 vs 終値高値4,940 / 髭安値3,180 vs 終値安値3,255
  - 9616: 髭高値3,785 vs 終値高値3,776 / 髭安値2,321 vs 終値安値2,356
- **1980**: 髭ベース値(3,680/1,433.3)は現在の株価水準(終値2,924円)と整合する桁
- **6637・9616(変更前=終値ベースとの差の記録)**:
  - 9616は高値+0.24%・安値-1.5%で「数%以内」の想定通り
  - **6637は高値+8.7%(4,940→5,370)と想定より大きい差**が出たため、生データで原因を裏取りした。2025-10-29に寄り付き5,370円→終値4,845円という大幅な窓開け後の反落(実データで確認)があり、この日の実際の高値(髭)が5,370円という事実そのものであることを確認。バグではなく、値動きの荒い1日が終値ベースと髭ベースの差を拡大させた正当な結果と判断
- **回帰確認**: `git stash`による変更前後比較で、EMA・現在値・財務系は52週値(w52_high/low/date)とnote以外すべてdiff一致

## 遡及の扱い(指示通り)
既存ノートは修正していない。1980ノートの52週値(終値ベース、当時の基準)はそのまま残し、次回更新時に自然に髭ベースへ移行する方針。
